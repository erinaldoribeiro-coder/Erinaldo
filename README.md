## 💻 Como Usar Localmente

1. Clone o repositório:
```bash
git clone https://github.com/erinaldoribeiro-coder/Erinaldo.git
```

2. Navegue até a pasta do projeto:
```bash
cd Erinaldo
```

3. Abra o arquivo `index.html` no seu navegador:
```bash
# No Linux/Mac
open "Gestão de Projetos/index.html"

# No Windows
start "Gestão de Projetos/index.html"
```

Ou simplesmente abra com um servidor local (recomendado):
```bash
# Usando Python 3
python -m http.server 8000

# Usando Node.js
npx http-server
```

Depois acesse: `http://localhost:8000/Gestão de Projetos/`
